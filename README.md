1. **Clone the repo**

   ```bash
   git clone https://github.com/jakkukarthik24/Cine-Hub.git
   ```

2. **Install dependencies**

   ```bash
   cd client
   npm install

   cd ../server
   npm install
   ```

3. **Set up environment variables**

   * Create a `.env` file in the `server/` directory:

     ```
     PORT=5000
     DB_URL=your_postgresql_connection_string
     ```

   ```bash
   cd server
   ```
